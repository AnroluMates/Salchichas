3. Potencias. Operaciones con potencias


Aunque las potencias de números  naturales ya las trabajasteis el curso pasado, queremos dedicarles un poco más de tiempo, sobre todo porque es necesario que ya aprendas a usar sus propiedades.

A continuación resumimos cómo calcular las potencias de números enteros.

 

Potencias de números naturales. ¿Cuál es la novedad?

Definición de Potencia de un número natural

Una potencia de un número natural es una forma abreviada de escribir una multiplicación de factores iguales.

Sabemos sus partes:  an = a·a·a·...·a (n factores).

Llamamos base a la a, y exponente a la n.

Ejemplo: (3)4 = 3·3·3·3 

Propiedad 1: Producto y cociente de potencias con la misma base.

Para multiplicar dos o más potencias con la misma base, se mantiene la misma base y se suman los exponentes:

Ejemplo: 63 · 65 = (6·6·6)·(6·6·6·6·6) = 6·6·6·6·6·6·6·6 = 65+3 = 68.

Para dividir dos o más potencias con la misma base, se mantiene la misma base y se restan los exponentes:

Ejemplo: 75 : 72 = (7·7·7·7·7) : (7·7) = 75-2 = 73 .

Propiedad 2: Potencias de exponentes 1 y 0

  • Una potencia de exponente 1 es igual a la base: a1 = a.
  • Una potencia de exponente 0 es igual a 1 SIEMPRE   a0 = 1.

Propiedad 3: Potencia de una potencia

Para elevar una potencia a otra potencia se mantiene la misma base y se multiplican los exponentes: (am)n = am

Propiedad 4: Potencia de un producto y de un cociente

  • La potencia de una multiplicación es igual a la multiplicación de las potencias: (a·b)n = an · bn.
  • La potencia de una división es igual a la división de las potencias; (a;b)n = an · bn
